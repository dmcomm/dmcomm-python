class dmcomm.protocol.core16.ResultSegment(sent: bool, data: int)¶

Describes the result of one segment of the communication for 16-bit protocols.

Parameters

  • sent – True if this represents data sent, False if data received.

  • data – A 16-bit integer representing the bits sent or received. If sent is False, can be None to indicate nothing was received before timeout.
